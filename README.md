## 功能特性

* 支持RSS订阅
* 支持Markdown语法
* 可定制文件上传目录
* 可定制网页META信息
* 可定制的XSS过滤策略
* 密码采用MD5进行传输，用BCrypt进行存储
* 上传的图片压缩存储
